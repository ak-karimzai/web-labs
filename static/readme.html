Goal Tracker Project

Цель работы

В контексте приложения для отслеживания личных целей и привычек Пользователь играет центральную роль.

Перечень функциональных требований

  1. Обеспечить Пользователю возможность создания, изменения и удаления личных целей.
  2. Обеспечить Пользователю возможность создания, изменения и удаления привычек, связанных с целями.
  3. Обеспечить Пользователю возможность регистрации и входа в систему.
